# MVP Scope

## Included

- Home dashboard
- SOS hold-to-activate mock flow
- Emergency countdown and emergency mode screens
- Trusted contacts CRUD
- Fake call simulation
- Appearance settings
- Activity log
- Demo mode
- Local Room and DataStore persistence
- Mock map and risk status cards

## Excluded For Now

- Real SMS alerts
- Real phone calls
- WhatsApp automation
- Real-time tracking
- Backend services
- AI prediction
- Danger heatmap
- Admin dashboard
- Background tracking service

The MVP uses simulated emergency alerts and demo location data. It does not send real emergency messages or replace official emergency services.
