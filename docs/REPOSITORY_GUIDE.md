# Kaval Repository Guide

Kaval is an Android-first personal safety companion MVP. The repository should show clear planning, clean code organization, honest demo status, and a scalable structure for future improvements.

The current focus is:

- Clean Android MVP
- Local-first mock safety flow
- Strong UI/UX direction
- Trusted contacts
- SOS flow
- Fake call simulation
- Appearance customization
- Activity logging
- Future-ready architecture

Kaval focuses first on building a reliable emergency interaction flow before expanding into real-time backend services and AI-assisted safety intelligence.
