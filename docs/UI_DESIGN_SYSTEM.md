# UI Design System

Design style: Protective Glass UI with Tactile Emergency Controls.

## Visual Principles

- Dark calm base
- Glassmorphism cards
- Subtle neomorphic buttons
- Psychological color coding
- High-contrast SOS button
- Accessibility-first layout

## Color Logic

- Red = Emergency
- Amber = Warning
- Green = Safe
- Blue = Trust and information
- Gray = Disabled or inactive
- Dark navy and charcoal = Background
- White and off-white = Text

Red is reserved for emergency actions and active alert states.
